How it works

1

Fetch

2

Parse & Analyze

AST parses imports. Radon calculates complexity. NetworkX builds the graph.

3

Embed

Code chunks → sentence-transformers embeddings → stored in ChromaDB

4

Chat

Your question → semantic search → top chunks sent to Claude → smart answer
